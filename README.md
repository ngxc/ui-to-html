## 🚀 解决方案：双模型编排
我们设计了一个“上传 UI 图片 → 自动生成 HTML/CSS”的工具。

你不需要会表达。
让 **AI 当你的嘴替**。

你不需要一个人干活。
让 **两个大模型当你的流水线员工**：

1. **视觉模型（Qwen3-VL）**：处理 UI 图片
   - 自动生成完整的 UI 设计描述
   - 包含组件位置、颜色、布局
   - 自动预留图片上传区域
   - 自动识别交互组件并预留接口

2. **代码模型（Qwen-Code 或其它 Code LLM）**：根据描述生成
   - HTML
   - CSS
   - 可选 JS（交互接口）

并且——支持持续对话、持续优化，完美适配领导各种莫名其妙的需求。

## 🔄 工具工作流程

1. **获取硅基流动 API Key**
2. **使用 Qwen3-VL 读取 UI 图片**
   - 自动生成 UI 的详细结构化描述
3. **将描述输入 Code LLM**
   - 自动生成 HTML/CSS 代码
4. **支持继续微调**
   - 例如：
     - “按钮往右一点”
     - “背景换蓝色”
     - “整体更扁平化”
   - 领导的奇怪需求？放心交给 AI。
